Introduction

Pop-Machina is a Horizon 2020 project that seeks to highlight and reinforce the links between the maker movement and circular economy in order to promote environmental sustainability and generate socio-economic benefits in European cities.

In this website I will be posting all the progress that I make in the pop-machina academy.

Project 1: Making the logo

In the following image you can see the logo that I have designed. The idea was to mix something that represents Santander with the concept of a space maker. Therefore, I have chosen to represent Santander as a wave because It is well known due to its beautiful bay. On the other hand, I have chosen a wrench, which is hidden inside the wave, to represent the space maker.

The logo has been designed usign the software Inkscape.

These are the steps that I have followed to make the logo:

  1. I have painted the wave using the tool "Pencil Tool": This allows you to creates freehand paths by drawing directly on the canvas.
  2. Once I've finished drawing the wave, I've used the tool "Node Tool" to manipulate the shape of the wave till I got the desired shape.
  3. I've written the text using the tool "Text Tool".
  4. After the wave and the text are finished, I've used a new layer to draw the wrench over the wave. To draw its shape I've repeated the steps 1 and 2.
  5. Finally, I've used the shortcut "ctrl+shift+R" to adjust the canvas to the size of the logo.
